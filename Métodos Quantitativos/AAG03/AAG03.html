Distribuio Bernoulli

A distribuio Bernoulli usada para modelar experimentos que possuem dois possveis resultados: "sucesso" (com probabilidade ($p$) ) e "fracasso" (com probabilidade ($1-p$) ).

A funo de probabilidade definida como:

$$ P(X = x) = \begin{cases} p & \text{se } x = 1, \\ 1-p & \text{se } x = 0 \end{cases} $$

Onde:

  • $0 \leq p \leq 1$: probabilidade de sucesso.
In[14]:
import numpy as np
import matplotlib.pyplot as plt

# Parmetros
p = 0.7  # Probabilidade de sucesso
n = 1000  # Nmero de simulaes

# Simulao
data = np.random.binomial(1, p, n)

# Visualizao
plt.hist(data, bins=2, rwidth=0.8, align='left', color='blue', alpha=0.7)
plt.xticks([0, 1], ['Fracasso (0)', 'Sucesso (1)'])
plt.title(f"Distribuio Bernoulli (p={p})")
plt.xlabel("Resultado")
plt.ylabel("Frequncia")
plt.show()

Distribuio Binomial

A distribuio Binomial descreve o nmero de sucessos em $n$ experimentos independentes, cada um com probabilidade de sucesso $p$.

A funo de probabilidade :

$$ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} $$

Onde:

  • $n$: nmero de experimentos.
  • $k$: nmero de sucessos.
  • $p$: probabilidade de sucesso em cada experimento.
In[19]:
from scipy.stats import binom

# Parmetros
n = 10  # Nmero de tentativas
p = 0.5  # Probabilidade de sucesso

# Resultados possveis
x = np.arange(0, n+1)

# PMF (Probability Mass Function)
pmf = binom.pmf(x, n, p)

# Visualizao
plt.bar(x, pmf, color='blue', alpha=0.7)
plt.title(f"Distribuio Binomial (n={n}, p={p})")
plt.xlabel("Nmero de sucessos")
plt.ylabel("Probabilidade")
plt.show()

Distribuio de Poisson

A distribuio de Poisson usada para modelar o nmero de eventos que ocorrem em um intervalo fixo de tempo ou espao, dado que os eventos ocorrem com uma taxa mdia conhecida $\lambda$ e independentemente.

A funo de probabilidade :

$$ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} $$

Onde:

  • $k$: nmero de eventos.
  • $\lambda$: taxa mdia de eventos.
In[4]:
from scipy.stats import poisson

# Parmetro
lam = 4  # Taxa mdia de eventos

# Resultados possveis
x = np.arange(0, 15)

# PMF
pmf = poisson.pmf(x, lam)

# Visualizao
plt.bar(x, pmf, color='blue', alpha=0.7)
plt.title(f"Distribuio de Poisson (={lam})")
plt.xlabel("Nmero de eventos")
plt.ylabel("Probabilidade")
plt.show()

Distribuio Geomtrica

A distribuio geomtrica modela o nmero de tentativas at o primeiro sucesso em uma sequncia de experimentos de Bernoulli independentes.

A funo de probabilidade :

$$ P(X = k) = (1-p)^{k-1} p $$

Onde:

  • $k$: nmero de tentativas at o sucesso.
  • $p$: probabilidade de sucesso em cada tentativa.
In[5]:

# Parmetro

plt.title(f"Distribuio Geomtrica (p={p})")
plt.xlabel("Tentativas at o sucesso")
plt.ylabel("Probabilidade")
plt.show()

Distribuio Uniforme

A distribuio uniforme contnua modela um intervalo $[a, b]$, onde todos os valores tm a mesma probabilidade.

A funo de densidade de probabilidade :

$$ f(x) = \begin{cases} \frac{1}{b-a} & \text{se } a \leq x \leq b, \\ 0 & \text{caso contrrio.} \end{cases} $$

Onde:

  • $a$: limite inferior.
  • $b$: limite superior.
In[8]:
from scipy.stats import uniform

# Parmetros
a = 0  # Limite inferior
b = 10  # Limite superior

# Gerar dados
data = uniform.rvs(loc=a, scale=b-a, size=1000)

# Visualizao
plt.hist(data, bins=20, color='blue', alpha=0.7, density=True)
plt.title(f"Distribuio Uniforme ({a}, {b})")
plt.xlabel("Valor")
plt.ylabel("Frequncia relativa")
plt.show()

Distribuio Exponencial

A distribuio exponencial modela o tempo entre eventos em um processo de Poisson.

A funo de densidade de probabilidade :

$$ f(x; \lambda) = \lambda e^{-\lambda x}, \text{ para } x \geq 0 $$

Onde:

  • $\lambda$: taxa mdia de eventos.
In[7]:
from scipy.stats import expon

# Parmetro

data = expon.rvs(scale=1/lam, size=1000)

# Visualizao
plt.hist(data, bins=20, color='blue', alpha=0.7, density=True)
plt.title(f"Distribuio Exponencial (={lam})")
plt.xlabel("Valor")
plt.ylabel("Frequncia relativa")
plt.show()

Distribuio Gaussiana (Normal)

A distribuio normal a mais conhecida e amplamente utilizada em estatstica, modelando muitos fenmenos naturais.

A funo de densidade de probabilidade :

$$ f(x; \mu, \sigma) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$

Onde:

  • $\mu$: mdia.
  • $\sigma$: desvio padro.
In[17]:
from scipy.stats import norm

# Parmetros
mu = 0  # Mdia
sigma = 1  # Desvio padro

# Gerar dados
data = norm.rvs(loc=mu, scale=sigma, size=1000)

# Visualizao
plt.hist(data, bins=20, color='blue', alpha=0.7, density=True)
plt.title(f"Distribuio Gaussiana (={mu}, ={sigma})")
plt.xlabel("Valor")
plt.ylabel("Frequncia relativa")
plt.show()
