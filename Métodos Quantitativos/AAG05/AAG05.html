Anlise de Regresso Linear Completa

Estudo de caso: Consumo de energia em funo da temperatura em um microcontrolador

Anlise completa de regresso linear, explorando a relao entre temperatura e consumo de energia em um microcontrolador. Este um cenrio comum em sistemas embarcados, onde o consumo de energia pode variar significativamente com a temperatura.

1. Gerao dos Dados

Gerando dados simulados que representam a relao entre temperatura ($x$) e consumo de energia ($y$). A relao ser modelada como:

$y = \beta_0 + \beta_1x + \varepsilon$

onde:

  • $\beta_0$ = 2 (intercepto base)
  • $\beta_1$ = 0.15 (inclinao)
  • $\varepsilon$ = rudo aleatrio normal

2. Anlise de Correlao

O coeficiente de correlao de Pearson ($r$) dado por:

$r = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n (x_i - \bar{x})^2 \sum_{i=1}^n (y_i - \bar{y})^2}}$

Vamos calcular e visualizar a correlao entre as variveis:

In[10]:
# Calcular correlao
correlacao = np.corrcoef(temperatura, consumo)[0,1]

# Criar scatter plot com linha de tendncia
plt.scatter(temperatura, consumo, color='blue', alpha=0.5, label='Dados')
plt.title(f'Relao entre Temperatura e Consumo\nCorrelao = {correlacao:.4f}')
plt.xlabel('Temperatura (C)')
plt.ylabel('Consumo (mA)')

# Adicionar linha de tendncia
z = np.polyfit(temperatura, consumo, 1)
p = np.poly1d(z)
plt.plot(temperatura, p(temperatura), 'r--', alpha=0.8, label='Tendncia')
plt.legend()
plt.show()

3. Estimao dos Parmetros

Os parmetros da regresso linear so estimados pelo mtodo dos mnimos quadrados:

$b_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}$

$b_0 = \bar{y} - b_1\bar{x}$

In[11]:
# Realizar regresso linear
X = temperatura.reshape(-1, 1)
reg = LinearRegression().fit(X, consumo)
b1 = reg.coef_[0]
b0 = reg.intercept_

# Calcular valores preditos e resduos
y_pred = reg.predict(X)
residuos = consumo - y_pred

print(f"Equao da reta: Consumo = {b0:.4f} + {b1:.4f} * Temperatura")
print(f"R = {reg.score(X, consumo):.4f}")

Equao da reta: Consumo = 2.5853 + 0.1330 * Temperatura
R = 0.8915

4. Anlise dos Resduos e Clculo de Erros Padro

O erro padro da estimativa dado por:

$s_e = \sqrt{\frac{\sum e_i^2}{n-2}}$

onde $e_i$ so os resduos.

In[12]:
# Calcular erro padro
n = len(temperatura)
mse = np.sum(residuos**2) / (n-2)
se = np.sqrt(mse)

# Calcular erro padro dos parmetros
x_mean = np.mean(temperatura)
sxx = np.sum((temperatura - x_mean)**2)
sb1 = se / np.sqrt(sxx)
sb0 = se * np.sqrt(1/n + x_mean**2/sxx)

print(f"Erro padro da estimativa (Se): {se:.4f}")
print(f"Erro padro de b0: {sb0:.4f}")
print(f"Erro padro de b1: {sb1:.4f}")

Erro padro da estimativa (Se): 0.4301
Erro padro de b0: 0.3595
Erro padro de b1: 0.0088

5. Intervalos de Confiana

O intervalo de confiana para os parmetros dado por:

$b_i \pm t_{[1-\alpha/2, n-2]}s_{b_i}$

onde $t_{[1-\alpha/2, n-2]}$ o valor crtico da distribuio t de Student.

In[13]:
def calcular_ic(b, sb, n, confianca):
    t_crit = stats.t.ppf((1 + confianca) / 2, n-2)
    margem = t_crit * sb
    return (b - margem, b + margem)

# Calcular ICs para diferentes nveis de confiana
for conf in [0.90, 0.95, 0.99]:
    ic_b0 = calcular_ic(b0, sb0, n, conf)
    ic_b1 = calcular_ic(b1, sb1, n, conf)
    print(f"\nIntervalos de Confiana ({conf*100}%):")
    print(f"b0: ({ic_b0[0]:.4f}, {ic_b0[1]:.4f})")
    print(f"b1: ({ic_b1[0]:.4f}, {ic_b1[1]:.4f})")
